## Fonts Used in StudyVault

This project utilizes the following fonts:

1. **Roboto**
   - Source: Google Fonts
   - Usage: Main body text and headings
   - Link: [Roboto on Google Fonts](https://fonts.google.com/specimen/Roboto)

2. **Open Sans**
   - Source: Google Fonts
   - Usage: Secondary text and UI elements
   - Link: [Open Sans on Google Fonts](https://fonts.google.com/specimen/Open+Sans)

### Installation

To use these fonts in your project, include the following links in the `<head>` section of your HTML files:

```html
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&family=Open+Sans:wght@400;600&display=swap" rel="stylesheet">
```

### Usage in CSS

You can apply these fonts in your CSS as follows:

```css
body {
    font-family: 'Roboto', sans-serif;
}

h1, h2, h3, h4, h5, h6 {
    font-family: 'Open Sans', sans-serif;
}
```
